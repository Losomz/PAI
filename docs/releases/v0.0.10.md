# PAI v0.0.10

发布日期：2026-06-09

## 更新亮点

- 修复 Windows 自动打包时更新签名私钥被错误按有密码方式解码的问题。

## 修复与优化

- 移除 GitHub Actions 中的 `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` 环境变量传递。
- 保持使用无密码 updater 私钥生成 `latest.json` 与 `.sig` 更新签名文件。

## 技术与兼容性说明

- 本版本主要修复 CI 发布流程；推送 tag 后会重新触发 Windows 自动打包流程。
- 打包成功后，Release Assets 应包含安装包、`latest.json` 和 `.sig` 文件，用于自动更新检测与验证。

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
