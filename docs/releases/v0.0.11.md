# PAI v0.0.11

发布日期：2026-06-09

## 更新亮点

- 重新生成无密码 updater 签名密钥对，修复 CI 自动打包签名失败的问题。

## 修复与优化

- 替换 `tauri.conf.json` 中的 updater 公钥，与新的无密码私钥配对。
- CI 构建不再需要 `TAURI_SIGNING_PRIVATE_KEY_PASSWORD`，简化发布流程。
- 打包成功后 Release Assets 应包含安装包、`latest.json` 和 `.sig` 签名文件，自动更新链路可完整运行。

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
