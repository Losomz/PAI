# PAI v0.0.1

发布日期：2026-06-06

## 更新亮点

- 🎉 **PAI 项目正式启动** — 基于 Tauri + Vue 3 构建的个人 AI 工具箱桌面应用。
- 🔄 **AgentFramework 工具同步** — 支持从 AgentFramework 拉取工具并自动提交到 Git 仓库。
- 📋 **同步记录管理** — 重构为同步记录管理模式，支持本地路径和远程仓库的全量/增量同步。

## 主要变更

### 项目初始化

- 搭建 Tauri 2.x + Vue 3 + TypeScript + Vite 项目框架。
- 引入前端路由（Vue Router）、基础布局与首页视图。
- 配置 Rust 后端模块骨架（`git`、`sync`），预留扩展能力。
- 添加 Windows 构建脚本 `build.bat`。

### 同步功能

- 实现 AgentFramework 工具同步核心逻辑，支持从远程获取工具并将其变更自动提交到本地 Git 仓库。
- 新增前端同步视图 `SyncView.vue`，提供可视化同步操作界面。
- 重构同步模块为**同步记录管理模式**：
  - 通过 `records.rs` 持久化记录每次同步的源路径、目标仓库与同步状态。
  - 支持**本地路径同步**与**远程仓库同步**两种模式。
  - 支持**全量同步**与**增量同步**策略。
- 同步操作与 Git 模块深度集成，变更自动暂存、提交并附带可追溯的提交信息。

## 修复与优化

- 初始版本，无历史修复记录。

## 技术与兼容性说明

- **桌面框架**：Tauri 2.x（Rust 后端 + WebView 前端），当前仅配置 Windows 构建。
- **前端**：Vue 3 Composition API + TypeScript + Vite。
- **包管理**：npm（前端）+ Cargo（Rust）。
- **最低兼容**：Windows 10+（当前仅提供 Windows 构建）。
