# PAI v0.0.12

发布日期：2026-06-09

## 更新亮点

- 修复 CI 构建步骤中签名私钥通过环境变量传递时 Base64 解码失败的问题。

## 修复与优化

- 移除 Build 步骤中的 `TAURI_SIGNING_PRIVATE_KEY` 环境变量，仅保留 `TAURI_SIGNING_PRIVATE_KEY_PATH` 文件路径方式读取私钥。
- 私钥文件由前置步骤从 GitHub Secret 写入磁盘，构建时直接从文件读取，避免环境变量对 Base64 内容的编码问题。

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
