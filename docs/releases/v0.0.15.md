# PAI v0.0.15

发布日期：2026-06-10

## 更新亮点

- 新增应用设置面板，支持用户手动检查更新，提升更新体验的透明度和可控性。

## 主要变更

- **设置面板入口**：侧边栏底部新增"设置"按钮，点击后在右侧弹出 Drawer 展示设置面板。
- **手动检查更新**：设置面板中提供"检查更新"按钮，用户可随时主动检查是否有新版本可用。若无新版本会提示"当前已是最新版本"，有新版本则弹出确认对话框引导下载安装。
- **更新逻辑重构**：`useUpdater` hook 支持静默/显式两种模式；启动时自动检查为静默模式（后台进行），手动触发为显式模式（提供用户反馈）。hook 现暴露 `checkForUpdate` 方法供外部组件调用。
- **自动更新迁移**：自动更新调用从 `App` 组件迁移至 `MainLayout`，与设置入口同层级管理，职责更清晰。

## 技术与兼容性说明

- 新增 `SettingsView` 组件，接收 `onCheckUpdate` 和 `downloading` props，便于在不同场景下复用。
- `MainLayout` 引入 `Drawer` 组件展示设置面板，不影响现有页面布局。
- 本次变更仅涉及前端 UI 和更新交互逻辑，不影响后端、打包流程或数据存储。
