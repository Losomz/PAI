# PAI v0.0.13

发布日期：2026-06-10

## 更新亮点

- 引入更新签名公钥文件，为 Tauri 自动更新链路提供完整的签名验证支持。

## 修复与优化

- 新增 `pai-update-private.key.pub` 公钥文件，客户端可据此验证更新包签名完整性。
- CI `windows-release.yml` 注入 `TAURI_SIGNING_PRIVATE_KEY` 和 `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` 环境变量，构建时同时支持文件路径和密钥内容两种签名方式。

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
