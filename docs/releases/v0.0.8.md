# PAI v0.0.8

发布日期：2026-06-09

## 更新亮点

- 🔄 同步功能增强：前端现在可以展示实际复制的文件数量，同步结果更直观
- 🛡️ Git 操作容错改进：克隆或拉取失败时自动清理残留缓存，下次操作会重新克隆，避免脏状态导致的持续失败

## 主要变更

- 📦 CI 自动打包流程增强：增加 updater 签名密钥验证、产物完整性校验，自动上传 `.sig` 签名文件和 `latest.json`，为自动更新做好准备
- 🔐 新增 `process:default` 权限，开启 `createUpdaterArtifacts`，支持自动更新产物生成

## 修复与优化

- 同步源目录为空时明确报错，避免静默失败
- 版本号统一至 `0.0.6-beta.1`（Cargo.toml / tauri.conf.json）

## 技术与兼容性说明

- 本次更新开启了自动更新产物创建，后续版本可通过 Tauri updater 实现应用内自动更新
- CI 流程新增签名验证和产物校验步骤，确保发布包完整性

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
