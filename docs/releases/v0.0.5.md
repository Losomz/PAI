# PAI v0.0.5

发布日期：2026-06-08

## 更新亮点

无（本次为修复版本）。

## 主要变更

无功能新增。

## 修复与优化

- 🪟 **Windows Git 命令不再弹出控制台窗口** — 在使用 Git 子进程执行命令（如克隆仓库、提交等操作）时，原来会弹出短暂的控制台窗口，现已通过添加 `CREATE_NO_WINDOW` 标志隐藏，提升 Windows 桌面端用户体验。

## 技术与兼容性说明

- **修复范围**：仅影响 Windows 平台 (`src-tauri/src/git/mod.rs`)，Linux/macOS 不受影响。
- **框架版本**：React 18 + Tauri 2.x，与前版本保持一致。
- **最低兼容**：Windows 10+。

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
