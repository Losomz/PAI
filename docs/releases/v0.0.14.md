# PAI v0.0.14

发布日期：2026-06-10

## 更新亮点

- 重建 Tauri 自动更新签名流程，解决 Windows 自动打包在 updater 私钥解码、密码提示和产物校验阶段反复失败的问题。

## 修复与优化

- 重新生成无密码 Tauri updater 签名密钥对，并同步更新 `tauri.conf.json` 中的 updater 公钥。
- GitHub Actions 构建改为 `tauri build --ci`，避免无密码签名私钥在 CI 中触发交互式密码提示。
- CI 只通过 `TAURI_SIGNING_PRIVATE_KEY` 读取私钥，并显式使用空密码；移除文件路径和 GitHub Secret 密码变量的混用。
- 删除仓库中容易混淆的旧公钥文件，避免将 PEM/RSA 公钥或 `.pub` 文件误用于 updater 私钥。
- 新增 updater 签名配置说明文档，明确无密码私钥、GitHub Secret 和公钥配置的关系。

## 技术与兼容性说明

- Windows Release workflow 现在会在构建后根据 `.sig` 和 Release asset URL 生成 `latest.json`，确保 GitHub Release Assets 中包含自动更新所需的 manifest。
- 已通过 `workflow_dispatch` 手动验证 Windows 打包流程可成功运行，再通过本 tag 触发正式自动打包。

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
