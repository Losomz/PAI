# PAI v0.0.4

发布日期：2026-06-08

## 更新亮点

- 🏷️ **首页版本号显示** — 应用首页底部新增版本号展示，方便用户和测试人员快速确认当前运行版本。

## 主要变更

### 前端

- `HomeView` 首页新增 `getVersion()` 调用，通过 Tauri App API 读取应用配置中的版本号，并在页面底部以灰色小字显示（格式：`v0.0.4`）。版本号仅在成功获取后渲染，加载中不显示占位内容。

### 配置

- `tauri.conf.json` 中应用版本号从 `0.1.0` 修正为 `0.0.3`，与项目历史 tag 保持一致（本次发布推进至 `0.0.4`）。

## 修复与优化

- 无历史修复（本次为功能新增与配置修正）。

## 技术与兼容性说明

- **前端框架**：React 18（TypeScript + Vite）。
- **桌面框架**：Tauri 2.x（Rust 后端 + WebView 前端），当前仅配置 Windows 构建。
- **版本号来源**：通过 `@tauri-apps/api/app` 的 `getVersion()` 从 `tauri.conf.json` 读取，无需手动维护多处版本字符串。
- **最低兼容**：Windows 10+（当前仅提供 Windows 构建）。

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
