# PAI v0.0.9

发布日期：2026-06-09

## 更新亮点

- 修复 Windows 自动打包流程中更新签名私钥未传递的问题，确保自动更新产物可以正常生成。

## 修复与优化

- 在 GitHub Actions 的 Tauri 构建步骤中显式传入 `TAURI_SIGNING_PRIVATE_KEY`。
- 保留签名私钥文件路径与密码环境变量配置，提升不同 Tauri CLI 签名读取方式下的兼容性。

## 技术与兼容性说明

- 本版本主要修复发布流水线问题；推送 tag 后应重新触发 Windows 打包，并生成 `latest.json` 与 `.sig` 更新签名文件。

## 下载说明

请在本 Release 的 Assets 中下载 Windows 安装包或可执行文件。
