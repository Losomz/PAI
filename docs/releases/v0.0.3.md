# PAI v0.0.3

发布日期：2026-06-08

## 更新亮点

- 🛡️ **全量同步二次确认** — 全量同步按钮增加防误操作弹窗，避免意外覆盖本地文件。
- ⚡ **同步命令异步化** — 后端同步操作改为异步执行，不再阻塞主线程，GUI 响应更流畅。

## 主要变更

### 同步模块优化

- 前端：在 `SyncView` 中为全量同步按钮添加 Popconfirm 二次确认弹窗，用户需确认后才会执行全量同步，防止误操作导致文件丢失。
- 后端：`sync_direct` 命令从同步改为异步，使用 `tokio::spawn_blocking` 将耗时操作移出主线程，避免界面卡顿或无响应。
- 后端：`log_action` 日志输出从 stdout 改为 stderr，遵循标准日志输出约定，避免干扰正常管道输出。

### 依赖更新

- `Cargo.toml` / `Cargo.lock`：新增 `tokio` 异步运行时依赖，为后续并发操作提供基础。

## 修复与优化

- 无历史修复（本次为优化性重构）。

## 技术与兼容性说明

- **前端框架**：React 18（TypeScript + Vite）。
- **桌面框架**：Tauri 2.x（Rust 后端 + WebView 前端），当前仅配置 Windows 构建。
- **异步运行时**：新增 tokio 依赖，后续同步和 Git 操作均可受益于异步调度。
- **最低兼容**：Windows 10+（当前仅提供 Windows 构建）。
